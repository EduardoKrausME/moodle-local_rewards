# local_rewards

A Moodle local plugin that connects digital rewards to activity completion.

## Main features

- Reward section inside every activity settings form
- Badge bank with reusable name, description, and image
- Optional custom reward per activity
- Automatic issue when the activity is completed
- Full screen popup on the next page load
- Dedicated reward page
- Public share page
- LinkedIn share button using the public page URL
- Personalized SVG share image
- Daily repair task for missed completion events

## File areas

- `badgeimage`: reusable bank images
- `configimage`: activity specific images

## Main tables

- `local_rewards_badges`
- `local_rewards_configs`
- `local_rewards_issues`

## Notes

- The LinkedIn button uses the public page URL through the offsite share endpoint, while the public page itself is responsible for the visible content and image preview.
- The personalized image is generated by `issue_image.php` as SVG.
- The first release grants rewards only from activity completion, matching the requested scope.

## Suggested next improvements

- Open Graph tags on the public page for stronger social previews
- Per activity background frames for the personalized image
- Teacher reports for issued rewards
- Optional manual revocation flow
